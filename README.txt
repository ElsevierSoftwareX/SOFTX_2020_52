CNAplot release 1.2, Mac OS X (32 bit), Ubuntu (64 bit) and Windows 10 (32 bit). 

ChartDirector license included in compiled binaries, not in source code. Note: The source code can be compiled without the license. 

Important: You may need to control-click to open the software in OS X and choose to accept that this software comes from unknown developer. In Windows 10 you may get a Windows Defender warning when launching the first time. Choose to open it, if you want to try out the software. There is no need to install anything on either platform. 

New in this release: Centromer positions are preloaded in software (UCSC tablebrowser GRCh37/38) and shown on plot. Positions on chromosome p- and q-arm are included in read depth ratio U-tests.
